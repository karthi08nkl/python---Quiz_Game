# Quiz Game

A beginner-friendly **Quiz Game** built in Python that challenges players with multiple-choice questions through a command-line interface. The game validates user input, calculates the final score and percentage, provides performance-based feedback, and allows players to replay the quiz.

## Features

* Multiple-choice quiz questions
* Input validation for answer selection
* Instant feedback for correct and incorrect answers
* Final score and percentage calculation
* Performance-based result messages
* Replay option to play the quiz multiple times
* Interactive command-line interface

## Technologies Used

* Python 3

## Python Concepts Used

* Functions
* Lists
* Loops (`for` and `while`)
* Conditional Statements (`if`, `elif`, `else`)
* Variables and Data Types
* User Input Validation


Created as a Python practice project.
